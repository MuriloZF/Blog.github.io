Lógica Algorítmica

A matéria Lógica Algorítmica foi a primeira em que aprendemos a programar, através da linguagem de programação Python.

Uma das peculiaridades dessa matéria foi a tão temida prova de programar no papel.

Nessa matéria também foi desenvolvido em equipes um jogo de Jokenpo com três modos de jogo:
1- Jogador vs Jogador
2- Jogador vs Computador
3- Computador vs Computador

O último projeto da matéria foi desenvolver um jogo de Batalha Naval.
